## **âš™ï¸ Setup & Installation**
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/yourusername/maia.git
   cd maia
   ```

2. **Create a Virtual Environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   ```

